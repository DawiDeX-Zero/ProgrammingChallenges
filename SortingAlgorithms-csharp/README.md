# Sorting Algorithms
Implementation of popular sorting algorithms.
## Available algorithms
1. Insertion
2. Bubble
3. Selection
4. Merge
5. Counting
6. Bucket
7. QuickSort
8. Heap
## Usage
```
<name of algorithm>.SortAscending(List<int>)
or
<name of algorithm>.SortDescending(List<int>)
```
